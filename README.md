# **PROYECT SETUP**

## **CLONE THE REPOSITORY**

```
git clone https://github.com/Allorentee/FeelTheArt-WebPage.git
```

# **BACKED**

## **PUT INTO BACKEND FOLDER**

```
cd FeelTheArt-Back
```

## **INSTALL THE DEPENDENCES**

```
npm i
```

## **KEEP COMPILATION**

```
npm run build:watch
```

## **KEEP SERVER STARTED**

```
npm run start:watch
```
